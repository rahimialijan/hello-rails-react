# 📖 "Hello world" project as one app with Webpack<a name="about-project"></a>

**[Description]**

This project aims to set up a "Hello world" application utilizing both Ruby on Rails as the back-end and React as the front-end. You'll have the chance to understand different approaches to connect Ruby on Rails back-end with React front-end.

## 🛠 Built With <a name="built-with"></a>

### Tech Stack <a name="tech-stack"></a>

- Ruby on Rails >=7
- React
- Webpack
- Postgres

### Key Features <a name="key-features"></a>

- Connection between Ruby on Rails and React
- API endpoint for random greetings
- Usage of Webpack and jsbundling-rails gem
- CRUD operations on Postgres database

## 💻 Getting Started <a name="getting-started"></a>

### Prerequisites

- [Ruby on Rails >=7](https://rubyonrails.org/)
- [React](https://reactjs.org/)
- [Webpack](https://webpack.js.org/)
- [Postgres](https://www.postgresql.org/)

### Install

```
git clone https://github.com/rahimialijan/hello-rails-react.git
cd hello-world-webpack
bundle install
npm install
rails s
```
